# Who Speaks When? Contribution Fairness in Federated Learning With Missed Model Updates
# Abstract
This paper addresses contribution fairness in federated learning environments characterized by intermittent client connectivity and missed global model updates. To achieve equitable participation, we propose FEDFresh, a probabilistic participation policy that regulates decentralized transmissions to ensure users exert an equal expected impact on the learned model.
# Repository Explanation
This repository contains the code for reproducing the results of the paper and extended mathematical proofs for Theorem III.1 regarding the asymptotic expected contribution rate. The proof details the application of renewal reward theory to model user availability cycles, ultimately deriving the closed-form expressions for the optimal transmission probabilities necessary to guarantee uniform long-term contribution across the network.
